# 校正機器管理システム (calibrationmanager)

Java (Spring Boot) で開発した、製造現場や研究所向けの校正管理システムです。

##  主な機能
* **校正機器の管理:** 機器の登録・編集・削除機能（CRUD機能の実装）
* **次回校正時期の自動計算:** 校正日と周期に応じた自動計算（月末自動調整ロジックの組み込み）
* **校正の合否判定:** 判定結果に応じたステータスの自動更新
* **校正履歴の管理:** 過去の校正履歴一覧の自動記録・保持

##  使用技術
* **バックエンド:** Java 17 / Spring Boot 3
* **フロントエンド:** HTML5 / Thymeleaf / CSS3
* **データベース:** H2 Database (開発用インメモリDB)

##  こだわったポイント
* **業務の効率化を意識した自動計算ロジック**
  校正日を入力するだけで次回の校正期限が自動計算されるため、ユーザーが手動で計算・入力する手間を完全に省き、ヒューマンエラーを防止します。
* **判定結果に連動したステータス管理**
  校正の合否結果（合格/不合格）に応じて、機器のステータスが自動的に「有効」または「使用停止」へと切り替わる仕組みをバックエンド側で制御し、現場での誤用を防ぐ堅牢な設計にしました。
* **画面遷移時のユーザー体験 (UX)**
  校正入力画面において「今どの機器を操作しているか」が一目でわかるよう、親切なコンテキスト情報の表示を追加し、誤操作のリスクを徹底的に排除しました。
